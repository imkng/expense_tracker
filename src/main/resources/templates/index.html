User Manual

Requirements

  • Postman Or You Can Use Swagger Documentation
  • For POST URL's:- Body -> raw -> Json
  • For Get/PUT/DELETE URL's:- Authorization -> Bearer Token -> Add token from Login

Brief Intro about the services

  1. 1. Registration
  2. 2. Login (You will get bearer token for auth.)
  3. 3. CRUD Operations on Expenses

Swagger Documentation>

REST API's

  1. Registration: http://3.111.121.31:5000/api/v1/register (POST)
  2. Login: http://3.111.121.31:5000/api/v1/login (POST)
  3. Create Expense:- http://3.111.121.31:5000/api/v1/expenses (POST)
  4. Read Expense:- http://3.111.121.31:5000/api/v1/expenses (GET)
  5. Update Expense:- http://3.111.121.31:5000/api/v1/expenses/{expense_id} (PUT)
  6. Delete Expense:- http://3.111.121.31:5000/api/v1/expenses/{expense_id} (DELETE)
